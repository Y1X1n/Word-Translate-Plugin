上传到 GitHub 的说明
==================

文件夹内容：
- 这个文件夹包含了所有需要上传到 GitHub 的项目文件

上传步骤：
1. 访问 https://github.com/Y1X1n/Word-Translate-Plugin
2. 点击 "Add file" -> "Upload files"
3. 将这个文件夹中的所有文件和文件夹拖拽到上传区域
4. 提交信息填写：Initial commit
5. 点击 "Commit changes"

上传完成后，GitHub Actions 会自动开始构建项目。
构建完成后可以在 Actions 页面下载生成的文件。
